# Skill Idea File Format

Each skill idea gets its own markdown file.

Path: `skill-ideas/ideas/`

Required sections:

```markdown
# [Idea Name]

## Problem

## Pain Point

## Description

## Value Driver
```

Recommended optional section:

```markdown
## Optional Notes
- Users:
- Current workaround:
- Success signal:
- Scope boundaries:
```

Keep files short and readable. Start rough, refine later.
