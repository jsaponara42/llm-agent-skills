# Skill Ideas Folder

This folder stores one markdown file per skill idea.

## Naming

Use:

`YYYY-MM-DD-short-kebab-name.md`

## How to add a new idea

1. Copy `skill-ideas/IDEA_TEMPLATE.md`
2. Save it in this folder with the naming format above
3. Fill in Problem, Pain Point, Description, and Value Driver
4. Add a link to the new file in `skill-ideas/SKILL_IDEAS.md`
