# Skill Idea Description Coach

## Problem
Skill ideas are easy to think of but hard to write down clearly and consistently.

## Pain Point
Ideas get lost in chats, sticky notes, and half-written docs, which slows down execution.

## Description
Create a lightweight workflow that turns rough brainstorms into clean one-file skill ideas using Problem, Pain Point, Description, and Value Driver.

## Value Driver
Faster capture means more ideas preserved, easier prioritization, and less startup friction for future skill development.

## Optional Notes
- Users: Solo builder, small product team
- Current workaround: Ad hoc notes across multiple tools
- Success signal: 5+ clear ideas captured per week with consistent format
- Scope boundaries: Not a full implementation planner
