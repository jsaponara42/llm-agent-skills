# Skill Idea Question Bank

Use only when the user needs help sharpening a rough idea.

Ask up to 3 of these:

1. What exact task feels slow, repetitive, or frustrating?
2. Who feels this pain most often?
3. What are they doing manually today?
4. What output should the skill produce every time?
5. What would make you say this skill is successful after one week?
6. What should this skill explicitly avoid doing?

Keep questions short and plain-language.
